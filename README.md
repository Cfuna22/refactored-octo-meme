## 🚀 Getting Started
Clone the repo and run locally:

```bash
git clone https://github.com/Cfuna22/refactored-octo-meme.git
cd portfolio
pnpm install
pnpm run dev
````

Then open [http://localhost:5173](http://localhost:5173) in your browser.
